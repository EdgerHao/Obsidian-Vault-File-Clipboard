# Vault File Clipboard 0.0.1

Initial standalone release.

## Features

- Copy files and folders from the Obsidian file explorer to the system clipboard.
- Paste external files and folders into a selected vault folder.
- Use `Command/Ctrl+C` and `Command/Ctrl+V` with file explorer selections.
- Drag vault files and folders to desktop applications with `Alt` + drag.
- Copy selected items to an optional configured target folder.
- Avoid overwriting existing vault files by generating numbered names.
- Native desktop support for macOS, Windows, and Linux.

## BRAT assets

This release includes `main.js`, `manifest.json`, and `styles.css` as individual assets for BRAT installation.
